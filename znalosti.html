Znalosti

Jakožto full-stack vývojář znám frontendové i backendové technologie. Z backendových technologií rozumím hlavně Javě a Nodejs. Z frontendových technologií znám HMTL, CSS, Javascript a React. Pracuji jako techlead, tedy mám zkušenosti s vedením lidí. Líbí se mi vývoj v agilním prostředí. Nebojím se řešit problémy. Hovořím anglicky na úrovni C1 a německy na úrovni B2.

Hard skills

Hard skills jsou pro programátora nutností. U mne se jedná především o vývoj v Javě a Javascriptu. Umím také HTML5, CSS3 a React. Mám znalosti návrhových vzorů a patternů. Umím používat vývojové technologie jako git, Linux a Docker.

Moje hlavní zaměření je jazyk Java. Javu používám aktivně v práci posledních 5 let. Kromě Java standart edition umím pracovat jak s frameworkem Spring boot, tak i s Jakartou Enterprise Edition. Dále umím používat Hibernate a psát více vláknové aplikace. Pro sestavení používám nástroje Maven a Gradle. V Javě pracuji na backendových technologiích webových aplikací. Javu vyvíjím výhradně v IntelliJ IDE, bez něhož se neobejdu!

Dalším jazykem který ovládám na vyšší úrovni je Java script. Vyznám se ve frontend frameworku React, ale mám přehled i o backendu, tedy o Nodejs. Java script aktivně používám poslední 3 roky. Mám zkušenost i s technologií Jquery.

Jakožto full-stack vývojář mám přehled i o frontendu. Umím HTML a CSS a jejich frameworky a rozšířeních jako jsou Bootstrap, Sass, Less. Když mě netlačí čas, tak raději šáhnu po čistém CSS a stránku si napíšu celou podle sebe. Také ovládám Google Chrome developer tools.

Z databázových technologií ovládám Oracle SQL (mám Oracle certificate). Z relačních databází umím ještě používat Postgress nebo MySQL. Mám také zkušenosti s prací s nerelačními databázemi, například Mongo DB. Hojně používám pattern object related mapping.

Mám základní znalosti programování, ke kterým patří objektově orientované programování - dědičnost, polymofismus, zapouzdření. Znám patterny REST a SOAP. Webové aplikace často programuji jako MVC. Mám i menší znalosti funkcionálního programování.

Mezi další technologie, kterým rozumím patří git, GitLab, GitHub, Bitbucket. Znám věci jako CI, CD a Docker. Jsem nadšený Linuxák, ale nemám problém pracovat na Mac Os nebo Windows. Umím psát Unit testy. Dále znám technologii Elastic search.

Soft skills

Pracuji jako techlead, tedy soft skill opravdu potřebuji. Musím umět řešit problémy. Umím vést lidi a pracovat s nimi. Nedílnou součástí mé práce je komunikace v cizích jazycích.

Jsem nekonfliktní člověk, problémy radši řeším v klidu bez zbytečných hádek. To neznamená, že problémy přehlížím. Je důležité se pravdě podívat do očí a pokusit se nalézt co nejlepší řešení. Jako programátor se s řešením problémů potkávám každý den a nezáleží na tom, jestli jde o problém v kódu nebo v týmu.

Mám zkušenosti s vedením týmu. Lidi si nevybírám jen podle jejich technického stacku, ale dbám na to, aby v týmu fungovala chemie, protože tým musí spolu dobře komunikovat. Mám rád agilní přístup. Dokážu v lidech najít potenciál a do budoucna ho rozvíjet.

Velký důraz kladu na efektivitu. Dokážu si dobře zorganizovat čas. Nerad plýtvám časem, kterého máme jen omezené množství. Hlavní je stanovit si priority a těm se věnovat. Přesto jsem perfekcionista a pokud to věci vyžadují, tak jim věnuji dostatečné množství času, aby výsledek stál opravdu za to.

Hovořím plynně česky. Angličtinu mám na úrovni C1, zvládám komunikaci jak mluvenou, tak písemnou. Získal jsem certifikát CAE. Německy hovořím na základní rovni (B2).
